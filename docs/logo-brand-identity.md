## Logo Concepts

### Concept 1: The Crown Diamond

**Visual:** A stylized diamond that doubles as a crown

**Elements:**
- Diamond shape with crown silhouette
- Facets suggest multiple hands reaching up
- Negative space creates "B" letterform

**Symbolism:**
- Diamond = Value, treasure, premium
- Crown = Winning, royalty, top bidder
- Facets = Community, many bidders

**Variations:**
- Full: Diamond-crown with wordmark
- Icon: Diamond-crown alone
- Animated: Diamond facets sparkle, crown rises

---

### Concept 2: The Winning Hand

**Visual:** Multiple hands reaching toward a central diamond

**Elements:**
- 3-5 stylized hands in circular formation
- Diamond floating in center
- Hands create circular frame

**Symbolism:**
- Hands = The crowd, community, competition
- Diamond = The prize, the deal, the treasure
- Circle = Unity, completion, safety

**Variations:**
- Full: Hands + diamond + wordmark
- Icon: Hands framing diamond
- Animated: Hands reach, one "wins" the diamond

---

### Concept 3: The Spark Bid

**Visual:** A diamond with a spark/burst emanating from center

**Elements:**
- Geometric diamond (low poly style)
- Burst/spark lines from center
- Lines form subtle "B"

**Symbolism:**
- Diamond = The item, the value
- Spark = The bid, the excitement, the win
- Burst = Energy, Nike-inspired motion

**Variations:**
- Full: Spark diamond + wordmark
- Icon: Diamond with dynamic lines
- Animated: Spark pulses, lines radiate

---

### Concept 4: The Treasure Map Pin

**Visual:** Map pin icon with diamond inside

**Elements:**
- Location pin shape
- Diamond embedded in pin head
- Subtle path lines (treasure map style)

**Symbolism:**
- Pin = Local, meetup, location
- Diamond = Treasure, value
- Path = The journey, the bidding process

**Variations:**
- Full: Pin + diamond + wordmark
- Icon: Pin with diamond
- Animated: Pin drops, diamond sparkles

---

## Recommended: Concept 3 â€” The Spark Bid

## Logo Usage

### Clear Space
- Minimum clear space: 1x diamond height on all sides
- No text or graphics within clear space

### Minimum Size
- Digital: 32px height
- Print: 0.5 inch height

### Backgrounds
- Preferred: White or Midnight Navy
- Acceptable: Light gray (#F5F5F5)
- Avoid: Busy patterns, clashing colors

### Incorrect Usage
- Don't stretch or distort
- Don't change colors
- Don't rotate
- Don't add effects (drop shadow OK for contrast)
- Don't place on busy backgrounds

---

## Animation Guidelines

### App Launch
1. Diamond fades in (200ms)
2. Spark pulses once (300ms)
3. Wordmark reveals (200ms)
4. Total: 700ms

### Bid Placement
1. Button press: Scale 0.95
2. Spark burst from button center
3. Diamond icon appears with sparkle
4. Success haptic

### Winning Animation
1. Screen dims slightly
2. Diamond scales up from center
3. Spark rays emanate outward
4. Confetti burst (subtle)
5. "You Won!" text reveals

### Loading States
- Diamond rotates slowly (2s per rotation)
- Spark pulses gently
- Gradient shifts subtly
