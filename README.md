# Biddt Project

A mobile marketplace application for iOS and Android with safe local exchange.

## Overview

**Name:** Biddt  
**Type:** Mobile marketplace app  
**Platforms:** iOS, Android  
**Key Features:**
- Marketplace browsing and search
- Bidding system with real-time updates
- Safe local exchange with QR code verification
- Seller analytics and earnings tracking
- In-app messaging/negotiation
- Secure checkout and payment

## Design Inspiration

**Primary Reference:** Nike app
- High-energy interactions
- Clean typography
- Premium visual feel
- Bold use of color and motion
